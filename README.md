## 소개

단순히 화면을 구현하는 것을 넘어 서비스의 전체 흐름을 이해하는 8년차 웹퍼블리셔입니다.  
PHP 기반 백엔드 로직과 Vue.js 프레임워크를 경험하며, 데이터가 화면에 반영되기까지의 전 과정을 실무에서 익혔습니다.

## 포트폴리오 구현 포인트

이 포트폴리오 자체가 퍼블리싱 실력을 보여주는 결과물입니다.

- 시맨틱 HTML5 구조 (`nav`, `section`, `article`, `footer`)
- CSS Custom Properties(변수)로 디자인 토큰 관리
- CSS Grid + Flexbox 반응형 레이아웃
- `IntersectionObserver`로 스크롤 페이드인 및 스킬 바 애니메이션
- 바닐라 JS 라이트박스 (이전/다음 · 키보드 · 터치 스와이프)
- `prefers-color-scheme` 다크모드 대응
- `prefers-reduced-motion` 애니메이션 접근성 처리
- WAI-ARIA (`aria-label`, `role="dialog"`, `aria-modal`) 적용
- `lang="ko"`, OG 태그, 메타 description 등 SEO 기초 작업
