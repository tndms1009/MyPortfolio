자기소개

단순히 화면을 구현하는 것을 넘어 서비스의 전체적인 흐름을 이해하는 퍼블리셔입니다. PHP 기반의 백엔드 로직과 Vue.js 프레임워크를 경험하며, 데이터가 화면에 반영되기까지의 전 과정을 실무에서 익혔습니다.

낯선 코드 구조 앞에서도 당황하지 않고 핵심을 분석해내는 속도가 빠르며, 기술적 접점에서 발생하는 이슈를 파악하여 개발팀과 유연하게 소통합니다. 고립된 개발자가 아닌, '소통의 유연함'을 통해 프로젝트의 완성도를 높이는 협업의 가치를 지향합니다.

현재에 안주하지 않고 React 등 최신 트렌드를 꾸준히 학습하며 기술적 유연함을 유지하고 있습니다.
